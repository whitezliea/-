SQL Server 在工厂中的核心作用

01 保存生产过程数据：工单、SN、Tag、质量结果、报警、日志。
02 支持历史追溯：通过 SN、orderId、batchId 查询完整生产过程。
03 支持 SCADA 报表：趋势、报警、产量、质量统计。
04 支持 MES 闭环：把检测结果回传到生产管理系统。
05 支持工业可靠性：队列、批量写入、离线缓存、补写、备份恢复。
06 支持长期运行：索引、分区、归档、慢查询优化。
SQL Server is the memory center of the factory software. It stores production data, supports traceability, and protects data when failures happen.
SQL SERVER = INDUSTRIAL MEMORY / TRACEABILITY CENTER · PLC = REAL-TIME CONTROL · SCADA / COLLECTOR = ACQUISITION & DISPLAY · MES = PRODUCTION MANAGEMENT
